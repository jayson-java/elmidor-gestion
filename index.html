Conecta tu Google Sheet

Esta app guarda todos tus datos en una Google Sheet propia. Sigue estos pasos una sola vez para conectarla.

1
Crea el Google Sheet

Con las 5 hojas: Produccion, Stock, Gastos, Ventas, Resumen. Ver guía incluida.

2
Instala el conector

Extensiones → Apps Script → pega el archivo AppsScript.gs → Implementar

3
Pega la URL aquí abajo

La URL que te entrega Apps Script al implementar como app web

¿No la tienes lista? Abre GUIA-INSTALACION.md incluida en la carpeta del proyecto.

Iniciar sesión

Seleccioná tu usuario e ingresá tu PIN.
